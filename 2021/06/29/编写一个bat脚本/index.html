编写一个bat脚本

Posted on 2021-06-29

编写一个bat脚本入门

有没有跟我一样一脸懵的同志，bat脚本是什么？？？

标准答案：上百度——批处理

​ bat脚本就是DOS批处理脚本，就是将一些DOS命令按照一定顺序排列而形成的集合，运行在Windows命令行环境上。

我的理解（略狭隘）：就像python编译器执行python（.py）脚本一样，windows系统的命令提示符不就是执行DOS命令的吗，DOS命令写在一个脚本里面不就是.bat文件了吗，这不就是一个bat脚本了……

O_O（又要学习DOS命令了）

一、批处理常用命令

1. 常量

  • %CD%：获取当前路径
  • %PATH%：获取命令搜索路径（自己体会）
  • %DATE%：获取当前事件
  • %TIME%：获取当前日期
  • %RANDOM%：获取0-32767之间的任意十进制数字
  • %ERRORLEVEL%：获取上一条命令执行结果码，执行成功返回0，执行失败返回1。

2.echo

输出结果（类似python中的print）

3.echo off/on

echo off：输出结果，此语句后所有运行的命令都不显示

echo on：输出结果，执行命令时显示

4. @

输出结果，放在句首，该行运行的命令都不显示

都是输出结果，有什么不同？

  • echo：bat脚本里面的echo语句（运行命令）在命令提示符显示

  • @ ：后面的echo语句（运行命令）不显示（管一行）

  • echo off：echo off后面的echo语句（运行命令）都不显示（管后面所有，@如果在他后面没有什么用）

  • 一般脚本都这么写，对比上面截图就知道为啥了

  • 测试代码

5. call

调用别的bat文件执行后 继续执行当前文件

6. pause

暂停处理当前bat文件，等待用户按任意键后继续执行

7. rem（可以用::代替）

rem命令后面的字符为解释行，相当于注释（类似python中的#，放在echo off后面才有用）

8. 变量

set命令设置windows环境变量。命令提示符set /?或help set查看用法。

”=“后面全部是字符串变量，注意双引号也是字符串（区分python设置字符串变量）

调用变量：%[变量]%

9. 字符串基本操作

  • 字符串截取 ：echo %[字符串变量]:~[获取字符串起始位置],[获取字符串结束位置（不包含结束位置）]%（类似于python字符串切片）

    • ~：取字符标识符
    • 必须使用变量
    • 字符串位置计数从0开始，包含起始位置但是不包含结束位置

  • 字符串替换：[新的变量]=%[旧字符串]:[想要被替换的字符]=[新字符串]%

10. 参数

%[1-9]表示参数（类似shell传参，python函数传参），命令提示符执行脚本时传递相应参数

%0：表示批处理命令本身

11. start：启动一个单独的窗口以运行指定的程序或命令

12. exit：退出cmd.exe程序或当前批处理脚本

  • /B：退出当前批处理脚本而不是cmd.exe。

13.cls：清除屏幕内容

二、略高级的命令if,goto,choice,for

1. if

  • if [not] [/I] "参数"=="字符串" command
  • if [not] exist [路径\]文件名 command
  • if errorlevel <数字> command

需要注意的是，bat脚本的if，else中使用()区分；python中使用冒号和缩进区分。

/I：进行比较的字符串不区分大小写

  • EQU - 等于
  • NEQ - 不等于
  • LSS - 小于
  • LEQ - 小于或等于
  • GTR - 大于
  • GEQ - 大于或等于

2. goto

跳到goto所指定的标号（:字符串定义，标号所在行不执行）处，一般和if配合使用

3. choice

可以让用户输入一个字符，根据用户的选择返回不同的errorlevel，和if配合使用（类似于python的input）

help choice或者choice /?

  • /C choices：指定要创建的选项列表。默认列表是 “YN”。

  • /N ：在提示符中隐藏选项列表。提示前面的消息得到显示，选项依旧处于启用状态。

  • /CS：允许选择分大小写的选项。在默认情况下，这个工具是不分大小写的。

  • /T timeout：做出默认选择之前，暂停的秒数。可接受的值是从 0到 9999。如果指定了 0，就不会有暂停，默认选项会得到选择。

  • /D choice ：在 nnnn 秒之后指定默认选项。字符必须在用 /C 选项指定的一组选择中; 同时，必须用 /T 指定 nnnn。

  • /M text：指定提示之前要显示的消息。如果没有指定，工具只显示提示。

  • /?：显示此帮助消息。

  • 注意:
    ERRORLEVEL 环境变量被设置为从选择集选择的键索引。列出的第一个选择返回 1，第二个选择返回 2，等等。如果用户按的键不是有效的选择，该工具会发出警告响声。如果该工具检测到错误状态，它会返回 255 的ERRORLEVEL 值。如果用户按 Ctrl+Break 或Ctrl+C 键，该工具会返回 0的 ERRORLEVEL 值。在一个批程序中使用 ERRORLEVEL 参数时，将参数降序排列。

  • 示例:
    CHOICE /?
    CHOICE /C YNC /M “确认请按 Y，否请按 N，或者取消请按 C。”
    CHOICE /T 10 /C ync /CS /D y
    CHOICE /C ab /M “选项 1 请选择 a，选项 2 请选择 b。”
    CHOICE /C ab /N /M “选项 1 请选择 a，选项 2 请选择 b。”

4. for

for [/d /r /l /f] %%i in (set) do command

可以在命令行help for >.\for.txt将for使用方法保存在当前路径下的for.txt文件中。

  • /d：如果（set）集中包含通配符，则指定与目录名匹配，而不与文件名匹配。
  • /r：递归查询指定目录下的匹配文件。默认使用当前目录。
  • /l：该集表示以增量形式从开始到结束的一个数字序列。因此，(1,1,5)将产生序列1 2 3 4 5，(5,-1,1)将产生序列(5 4 3 2 1)

三、注意有坑

  1. bat脚本有中文在命令行执行的时候乱码

    命令提示符编码格式是__ANSI__，脚本保存时注意如果不是ANSI编码，运行时就会乱码。

    • 另存脚本为ANSI编码格式

    • 在代码页开始声明编码格式是UTF-8

      chcp 65001

  2. ECHO处于关闭状态

    • echo后面跟的变量名称不对

      echo str2=%str:girl=boy%输出str2=……，并没有设置str2这个变量，如果将这行的echo改为set，不会报错。

    • bat脚本=两边最好不要有空格（这个不确定，但是没有空格一定是对的）
